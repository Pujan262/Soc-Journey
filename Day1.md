# Day 1 — How the Internet Works
IP Address - A unique address assigned to the devices connected to network
DNS - Domain Name System works like phone book, it maps human readable or host name like google to it's ip address in server that user requested
ROUTERS - It serves as network converter or protocol converter, It's primary function is to connect two different networks lan and wan and providing routing between them.
Client and Servers - A client is a device that request something and server is a device that respond to that request.
Packets - A small chunks of data that information is broken into small parts before being send across the internet.
