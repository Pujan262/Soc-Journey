Day 8- Basic Linux Commands

Command	What it does
Pwd 	 Shows where you currently are
ls	Lists files in current folder
ls -la	Lists files with full detail (permissions, hidden files)
Cd ~	Goes to home folder
mkdir soc-lab	Creates a folder named soc-lab
touch soc-lab/test.txt	Creates an empty file inside soc-lab
cat soc-lab/test.txt	Shows file contents
cp soc-lab/test.txt soc-lab/copy.txt	Copies the file
mv soc-lab/copy.txt soc-lab/moved.txt	Renames the copied file
rm soc-lab/moved.txt	Deletes the renamed file
rmdir soc-lab	Deletes the now-empty folder
